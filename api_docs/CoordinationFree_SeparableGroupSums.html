CoordinationFree_SeparableGroupSums

Related Doc: package root

abstract class CoordinationFree_SeparableGroupSums extends CoordinationFree_SubmodularDoubleGreedy

Class implementing CF-2g for separable sums objective functions, of the form F(X) = \sum_l g(\sum_{i\in S_l\cap X} w_l(i)) - \lambda|X|. See Appendix F of paper for more details.
